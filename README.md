🌿 Greenden Project – Tailwind CSS Based Responsive Website
This is a responsive website project named "Greenden" built using Tailwind CSS — a modern utility-first CSS framework.

Greenden weiste: https://koushika-raju.github.io/Greenden-tailwind/index.html

The project showcases a clean and structured frontend for a fictional plant-selling or eco-themed website. It is designed with mobile-first responsiveness, ensuring a smooth user experience across all screen sizes — from mobile phones to desktops.

🔧 Key Features Implemented
Here are some Tailwind CSS features and utilities used in the project:

Responsive Design
Tailwind’s sm:, md:, lg: breakpoints were used to make sure the layout adapts smoothly to different screen sizes.

Flexbox & Grid Utilities
Layouts like product sections and navigation bars are built using flex, grid, justify-center, items-center, and gap-* classes — enabling fast and clean layout control.

Custom Styling with Utility Classes
Instead of writing custom CSS, Tailwind's pre-defined classes like text-xl, bg-green-500, rounded-lg, p-4, m-2, and shadow-md were used for styling elements easily.

Hover and Transition Effects
Used classes like hover:bg-green-700, transition, and duration-300 to make buttons and images interactive and visually appealing.

Typography Utilities
Fonts were controlled using font-bold, text-gray-600, text-2xl, tracking-wide, and other typography-related classes.

Tailwind Config
A custom tailwind.config.js file is used to manage themes, colors, and extend styles — useful for scaling the project further.
